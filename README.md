# Clinic Management System API

ASP.NET Core Web API for managing doctors, patients, and appointments.

## Features

- Manage Doctors (CRUD)
- Manage Patients (CRUD)
- Appointment Booking System
- Update Appointment Status

## Technologies

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- C#

## Notes

This project is for learning backend development with .NET.
